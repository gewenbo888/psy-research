# Psy Protocol — An Independent Research Brief

A third-party technical and ecosystem analysis of Psy Protocol: a ZK-native L1
built on Plonky2 recursive proofs, Poseidon-over-Goldilocks state, sharded
PARTH execution, a shielded UTXO payment layer, and an x402-native AI agent
stack.

The brief reads the protocol as an outside analyst, drawn from a ~20-repository
codebase corpus and the bridge specification (v0.4.3). It covers the PARTH
architecture, the proving stack, the privacy rail, the bridge problem, the
x402 wedge and AI tower, a full stack inventory, and the open questions.
